Plugin Name :  Newsfeed
News Source :  https://newsapi.org/
Author      : Arif Shah
Version     : 1.0
Author URI: https://www.linkedin.com/in/ariflateefshah/
Description: News feed plugin fetch news from https://newsapi.org/.It provides shortcode which can be used inisde posts and pages.


Setup:

1) Install the Newsfeed plugin
2) Activate Newsfeed
3) Go to Newsfeed configuration Menu and add API key eg: 3bfcdbd012e1491abf06e0e62e425893
4) Use shortcode [news] wherever you want to.
5) You can add filterations and sorting my passing params to the [news] shortcode eg: [news topic=”bitcoin” sortBy="publishedDate"]
